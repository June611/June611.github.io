你好，我是智越

您好，我是智越，一名即将毕业的计算机专业学生。
当前技术栈包括：C++、Python、Django、Vue3、MySQL、Docker、Linux，以及深度学习方向。

我正在学习 Web 后端开发、机器学习、人工智能和软件系统架构设计等方向；
曾独立开发过两个 Web 项目，一个是文件共享系统，另一个是类似工单提交管理系统。在这些项目中，我深刻体会了软件开发从需求分析、业务建模、模块划分到接口设计、测试部署的完整流程。虽然现代技术栈为开发提供了强大支持，但复杂业务场景下的挑战仍然考验工程师的系统性思维和技术解决能力，例如高并发处理、高可用设计、线程同步、分布式一致性等问题。

我逐渐熟悉了 Python、JavaScript、TypeScript、LayUI、Django、Vue3、MySQL 和 Docker 等工具与框架，深感 ORM 等技术在开发效率提升中的重要价值。

我对计算机处理器、程序高效执行、自然语言处理（NLP）、机器学习以及大语言模型（LLM）推理加速领域尤为感兴趣。
通过阅读《深入理解计算机系统》，了解到 CPU 如何通过 SIMD、缓存、分支预测等技术优化程序执行，这些内容让我十分着迷。而 LLM 的兴起则让我深入思考人类语言与机器智能之间的界限与融合，例如机器如何理解语言？是否能真正拥有语言能力？面对大模型所需的高算力和能耗，是否存在类似迁移学习的轻量级解决路径？

我也希望时刻提醒自己：
生活比工作更重要。目标可以远大，但不要被期望压垮；追梦的同时，也请记得爱家人、珍惜当下。
